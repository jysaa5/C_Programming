<h4> 3) 기능(함수) </h4>
- menu: 메뉴 생성 <br>
- input: 데이터 입력 <br>
- sortingSelect: 정렬 선택 <br>
- printingList: 데이터 출력 <br>
- bubbleSorting: 데이터 정렬 -> 버블 알고리즘 <br>
